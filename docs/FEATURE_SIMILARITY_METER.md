# מד דמיון לחיפוש AI 🎯

## תיאור התכונה

הוספנו מד ויזואלי שמציג למשתמש כמה כל תוצאה קרובה לשאילתה שהוא חיפש בחיפוש AI.

## איך זה נראה?

כל תוצאה בחיפוש AI תכלול:

```
┌─────────────────────────────────────┐
│ רמת התאמה:              מצוין ✓   │
│ ████████████████████░░░░  91%      │
└─────────────────────────────────────┘
```

## 5 רמות דמיון

המערכת מציגה 5 רמות דמיון שונות:

### 🟢 מצוין (91-100%)
תוצאה מעולה - התאמה כמעט מושלמת לשאילתה

### 🟢 טוב מאוד (73-90%)
תוצאה טובה מאוד - התאמה גבוהה לשאילתה

### 🟡 טוב (55-72%)
תוצאה טובה - התאמה סבירה לשאילתה

### 🟡 בינוני (36-54%)
תוצאה בינונית - התאמה חלקית לשאילתה

### 🟠 נמוך (0-35%)
תוצאה חלשה - התאמה נמוכה לשאילתה

## מתי המד מופיע?

המד מופיע **רק** בחיפוש AI (חיפוש סמנטי).

הוא **לא** מופיע ב:
- חיפוש לפי פרק ומשנה
- חיפוש טקסט חופשי
- חיפוש לפי נושאים/תגיות

## איך זה עובד מאחורי הקלעים?

המערכת משתמשת ב-vector embeddings כדי למדוד את המרחק הסמנטי בין השאילתה לבין כל משנה.

- **מרחק קטן** = דמיון גבוה = תוצאה טובה
- **מרחק גדול** = דמיון נמוך = תוצאה פחות רלוונטית

הנוסחה:
```
אחוז דמיון = ((0.72 - מרחק) / 0.22) × 100
```

## יתרונות

✅ **אינטואיטיבי** - צבעים וטקסט ברור בעברית
✅ **ויזואלי** - פס התקדמות עם אחוזים
✅ **מדויק** - מבוסס על חישוב מתמטי של המרחק הסמנטי
