## ✅ בדיקות חיפוש (Search Tests)

### 1. חיפוש לפי פרק ומשנה
- [ ] בחר פרק א' ומשנה א' - וודא שמוצג תוצאה אחת
- [ ] בחר פרק ו' ומשנה י"א - וודא שמוצג תוצאה אחת
- [ ] בחר פרק כלשהו ו"כל המשניות" - וודא שמוצגות כל המשניות של הפרק

### 2. חיפוש חכם (Smart Search)
#### 2.1 חיפוש מדויק (Exact Match)
- [ ] הפעל את המתג "חפש מילים מדויקות"
- [ ] חפש מילה שקיימת (למשל: "אבות") - וודא תוצאות
- [ ] חפש ביטוי מלא שקיים - וודא תוצאות מדויקות

#### 2.2 חיפוש AI סמנטי
- [ ] כבה את המתג "חפש מילים מדויקות"
- [ ] חפש שאלה או נושא (למשל: "תורה ועבודה") - וודא תוצאות רלוונטיות
- [ ] וודא שהלואדר מופיע במהלך החיפוש

#### 2.3 ולידציות חיפוש חכם
- [ ] נסה להזין טקסט באנגלית - וודא הודעת שגיאה בעברית
- [ ] נסה לשלוח חיפוש ריק - וודא הודעת שגיאה

### 3. חיפוש לפי נושא (Tag Search)
- [ ] בחר נושא אחד מהרשימה - וודא תוצאות
- [ ] בחר מספר נושאים - וודא תוצאות משולבות
- [ ] השתמש בשדה החיפוש של נושאים - וודא שהסינון עובד
- [ ] לחץ על "הצג עוד" בקטגוריה - וודא שמוצגים נושאים נוספים

---

## ✅ בדיקות תוצאות (Results Tests)

### 4. תצוגת תוצאות
- [ ] וודא שמוצג מספר התוצאות הנכון
- [ ] וודא שהנושאים (tags) מוצגים בצבעים הנכונים
- [ ] לחץ על כפתור העתקה - וודא שהמשנה מועתקת ללוח
- [ ] וודא שהכפתור משנה לצ'ק ירוק אחרי העתקה

### 5. ניווט בין משניות
- [ ] בחר משנה בודדת (לא הראשונה והאחרונה)
- [ ] לחץ "משנה הבאה" - וודא מעבר למשנה הבאה
- [ ] לחץ "משנה קודמת" - וודא מעבר למשנה הקודמת
- [ ] בדוק משנה ראשונה - וודא שאין כפתור "משנה קודמת"
- [ ] בדוק משנה אחרונה - וודא שאין כפתור "משנה הבאה"

### 6. חיפוש על ידי לחיצה על נושא
- [ ] בתוצאות, לחץ על אחד מהנושאים (tags)
- [ ] וודא שמתבצע חיפוש חדש לפי הנושא שנבחר

## ✅ בדיקות UI/UX

### 7. ממשק משתמש כללי
- [ ] וודא שהלוגו מוצג כראוי בראש הדף ובפוטר
- [ ] וודא שאנימציות האותיות העבריות פועלות
- [ ] גלול למטה - וודא שכפתור "חזור לחיפוש" מופיע
- [ ] לחץ על כפתור "חזור לחיפוש" - וודא גלילה לראש הדף

### 8. רספונסיביות (Responsive)
- [ ] בדוק בדפדפן במסך רחב - וודא תצוגה תקינה
- [ ] בדוק במסך צר (סימולציה של מובייל) - וודא תצוגה תקינה
- [ ] וודא שכפתורי סוג החיפוש עוברים לשורות במסך קטן

## ✅ בדיקות ביצועים ושגיאות

- [ ] וודא שהלואדר נעלם אחרי שהתוצאות מוצגות
