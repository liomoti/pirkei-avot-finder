# pirkei-avot-finder
A Flask-based web application for searching and managing Mishnayot from Pirkei Avot. Features include chapter-based search, free text search, tag-based search, semantic AI search, and content management. Built with SQLAlchemy, PostgreSQL, and deployed on Heroku.
